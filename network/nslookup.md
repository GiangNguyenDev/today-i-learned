# Introduction

Nslookup is a command-line tool to find DNS records (Domain Name System) of a domain name or IP address.

Example:

```console
C:\>nslookup wikipedia.org
Server:  fritz.box
Address:  192.168.178.1

Non-authoritative answer:
Name:    wikipedia.org
Addresses:  2620:0:862:ed1a::1
          91.198.174.192
```
